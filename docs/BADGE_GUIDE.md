## üéØ Shields.io Tips

### Create Custom Badges
Visit [shields.io](https://shields.io) to create custom badges:

```
https://img.shields.io/badge/<LABEL>-<MESSAGE>-<COLOR>
```

**Example:**
```markdown
![Custom](https://img.shields.io/badge/My-Custom_Badge-purple?style=for-the-badge&logo=github)
```

### Add Logos
Use `?logo=` parameter with [Simple Icons](https://simpleicons.org) names:

```markdown
![With Logo](https://img.shields.io/badge/Vue-3.5-42b883?logo=vue.js)
```

### Color Options
- Hex: `?color=667eea`
- Named: `?color=blue`
- State: `?color=success` (green)
